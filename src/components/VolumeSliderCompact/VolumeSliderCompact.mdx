# Volume Slider component
This is the compact variation of the volume slider that takes up less room on the bar.

<Props of={VolumeSliderCompact}/>

## Simple example

VolumeSliderDemo component passes the volume to the child components and provides a callback to set the volume.
<Playground>
  <Component initialState={{ volume: 70 }}>
  {({ setState, state }) => (
    <VolumeSliderCompact 
      volume={state.volume} 
      onVolumeChanged={(volume) => {
        setState({ volume });
      }}
    />
  )}
  </Component>
</Playground>


## Flipped example
