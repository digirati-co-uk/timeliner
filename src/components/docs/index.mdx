# Timeliner documentation

## Installation
To install the development environment you will require `node@8.11.x` and `yarn` installed.

To install run: `$ yarn`

To start the main application run: `$ yarn start`

and your server will be available at [http://localhost:5000/](http://localhost:5000/)

To start this documentation application, run: `$ yarn docz:dev`

and the documentation will be available at [http://localhost:5001/](http://localhost:5001/)
