# Volume Slider Demo

VolumeSliderDemo component passes the volume to the child components and provides a callback to set the volume.
<Playground>
  <Component initialState={{ volume: 70 }}>
  {({ setState, state }) => (
    <VolumeSlider 
      volume={state.volume} 
      onVolumeChanged={(volume) => {
        setState({ volume });
      }}
    />
  )}
  </Component>
</Playground>
