# Bubble display
Handles the rendering of many bubbles and the scrubber. You need to pass a render function to use.

**Still some unknowns with how this will work, and its responsibilities**

import { Playground, PropsTable } from 'docz';
import BubbleDisplay from './BubbleDisplay';
import DEMO_POINTS from './BubbleDisplay.DemoPoints';
import SingleBubble from '../SingleBubble/SingleBubble';



<PropsTable of={BubbleDisplay} /> 

## Bubble Display Demo
<Playground>
  <BubbleDisplay 
    points={DEMO_POINTS}
    width={600}
    height={200}
    x={0}
    zoom={1}
  >
    {points=>points.map(bubble => <SingleBubble {...bubble}/>)}
  </BubbleDisplay>
</Playground>
